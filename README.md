The product and download website for [Elcro](https://github.com/rksaklani/elcro), an open-source, local-first AI coding IDE.

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Production build

```bash
npm run build
npm start
```
